## Bug Reporting
Please be sure that the bug is related with CommentMentions specifically, and it not related with another mod, mod combination, or Geode itself. Make sure your bug hasn't been reported and/or fixed already.

## Suggestions
Please make sure your suggestion hasn't been suggested and/or implemented already. Make sure to add labels to your suggestion that are related with what the suggestion is.

## PRs
Make sure you have properly tested the feature you are working in

All PRs must be targeting the `dev` branch if they're changing anything related to the code (e.g. new features, bug fixes, refactors, etc). README changes (or just any changes not related to code at all) don't have to target to `dev` and can target `main`
